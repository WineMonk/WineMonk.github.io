NPOI安装

首先科普一下什么是NPOI——

NPOI是指构建在POI 3.x版本之上的一个程序，NPOI可以在没有安装Office的情况下对Word或Excel文档进行读写操作。

NPOI是一个开源的C#读写Excel、WORD等微软OLE2组件文档的项目。

一、安装NuGet

先科普一下——

Nuget是一个.NET平台下的开源的项目，它是Visual Studio的扩展。在使用[Visual Studio](https://baike.baidu.com/item/Visual Studio/539453)开发基于.[NET Framework](https://baike.baidu.com/item/NET Framework/10686851)的应用时，Nuget能把在项目中添加、移除和更新引用的工作变得更加快捷方便。

Nuget是 ASP .NET Gallery 的一员。NuGet 是免费、开源的包管理开发工具，专注于在 .NET 应用开发过程中，简单地合并第三方的组件库。

当需要分享开发的工具或是库，需要建立一个Nuget package，然后把这个package放到Nuget的站点。如果想要使用别人已经开发好的工具或是库，只需要从站点获得这个package，并且安装到自己的Visual Studio项目或是解决方案里。

NuGet能更方便地把一些dll和文件(如jquery)添加到项目中,而不需要从文件中复制拷贝。

一般VS都会安装NuGet，但是你安装VS时带的什么版本，它就会一直是那个版本，简言就是不会自动升级，如果你直接使用自带的NuGet安装NPOI就会出现这种问题——

所以你就需要手动升级NuGet

第一步、卸载旧版NuGet——打开VS -> 打开菜单栏 “工具” - “扩展和更新” -> 选择 “NuGet Package Manager” -> 点击 “卸载”

第二步、重启VS

第三步、安装新的NuGet——打开VS -> 打开菜单栏 “工具” - “扩展和更新” -> 点击 “联机” -> 搜索NuGet -> 下载

听起来就像是“把大象放进冰箱需要几步？” —— 打开冰箱 -> 把大象放进去 -> 关上冰箱

二、安装NPOI

打开VS —> 打开菜单栏 “工具” — ”NuGet 程序包管理器“ — “管理解决方案的 NuGet程序包” —> 搜索 “NPOI” —>下载
